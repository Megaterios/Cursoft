Bienvenido al sitio oficial de documentación sobre el proyecto Cursoft.

Cursoft es un proyecto de software, todo un sistema de información disponible para administrar el curso de profundización de "Desarrollo de Software" del programa de Ingeniería de Sistemas de la Universidad Francisco de Paula Santander. Esta página web permite tener el acceso a la documentación y código fuente más reciente y actualizada del proyecto. Para descargar el código fuente puede hacerlo como un archivo ".zip" o ".tar.gz" desde esta web o puede realizar el proceso mediante el sistema de control de versiones distribuido GIT, el procedimiento es el siguiente: 

$ cd public_html/
$ git clone https://github.com/Megaterios/Cursoft.git

Si usted desea sincronizar todas las ramas del proyecto:

$ git checkout --track origin/test
$ git checkout --track origin/dev
$ git checkout --track gh-pages

Documentación

El equipo del proyecto Cursoft ha dispuesto toda la documentación del proyecto en este sitio web, para tener la retroalimentación de diferentes personas y para que sea una guía para los interesados en desarrollar proyectos de software. Si desea observar la documentación de clic en la siguiente carpeta:

¿Cómo nació Cursoft?

En la materia "Análisis y diseño de sistemas" se debe elaborar un proyecto de software, así nació Cursoft, bajo la asesoria y colaboración de la docente encargada de la materia, la doctora Judith Del Pilar Rodriguez Tenjo.

Autores y Contribuidores

Cursoft cuenta con el apoyo de Jhorman Pérez (@jhorapb), Abdul Laiseca (@AbdulLaiseca), Manuel Saravia (@manuelsaravia) y Yeison Vargas (@YeisonVargas).

Agracedimientos especiales a: Albert Orellano y Ana Montesinos quienes colaboran atentamente en el proceso.
